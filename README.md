# ChatGPT Google Doc Formatting Prompt

**When generating any `.docx` file, Google-Docs-ready text, or long-form written document, always apply these formatting rules by default:**

### **Fonts**

- Use **Montserrat** for *all* text (titles, headers, body, bullets, captions).
- Do not substitute with Cambria, Arial, Times, or any system default.

### **Headers**

- **Title Block:**
    - Montserrat, bold, blue, larger size (approx 16 pt)
    - Subtitle directly under it in blue Montserrat (approx 14 pt)
- **Heading 1:**
    - Montserrat, bold, blue, medium size (approx 13 pt)
- **Heading 2:**
    - Montserrat, bold, blue, slightly smaller (approx 12 pt)

### **Spacing**

- Line spacing must be **1.0**
- Paragraph spacing: **6pt space after** each paragraph
- No extra spacing before paragraphs
- Keep sections tight and clean like the Imbue proposal example

### **Bullets**

- Always use **circle bullets (•)**
- Never use hyphens or dashes as bullets
- Maintain proper indentation identical to Google Docs default bullet indentation

### **Colors**

- Use the same **blue header color** as the Jess × Imbue DevRel Consulting Proposal
    
    (approx RGB **32, 76, 141**, unless otherwise specified)
    

### **General Layout**

- Avoid unnecessary spacing, blank lines, or overly large margins
- Follow the structural hierarchy: *Title block → Overview → Scope → Section headers → Body text → Bullets → Terms → Questions*
